



# MRTK 가상환경 재활운동 프로그램
### 🖥️ 프로젝트 소개
mrtk를 활용하며 홀로렌즈와 Nreallight 플랫폼에서 동작하는 운동을 통한 재활치료 
프로젝트 입니다. hand tracking을 활용하여 UI를 동작하고 mpu센서를 활용하여 
움직임을 감지합니다.  또한 올바른 자세로 운동할 수 있도록 가이드와 목표지점을 
설정해주며 가상공간을 활용하여 사용자가 언제 어디서나 간단한 운동을 즐길 수 있도록 
해주는 프로젝트 입니다.
